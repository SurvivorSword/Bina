The base surface for panels, topic cards, and stat blocks.

```jsx
<Card interactive media="/assets/moonlit-hero.jpg">
  <h3>Diffusion models</h3>
  <p>Peaked Q3 2023…</p>
</Card>
```

`interactive` adds hover lift + glow border; `glow` keeps the glow always on (featured). `media` renders a 16:9 image at top. Set `padded={false}` to control padding yourself.
